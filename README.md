## Project1

```y = (a + tg(x)) * (b + ln(a)) * e^(-b)```

## Project3

## Project5

```f(x) = sin(2 * pi * x) / 1 + x```

Найти произведение вычисленных значений функции `f(x)`, дробная часть которых < `0.5`

## Project6

## Project15

- наличие билетов на поезд с номером `ХХХ`;
